# Οδηγίες Εγκατάστασης — ArcheoVault

## Βήμα 1: Δημιουργία Google Sheet

1. Ανοίξτε το [Google Sheets](https://sheets.google.com) και δημιουργήστε ένα νέο υπολογιστικό φύλλο
2. Μετονομάστε το σε **"ArcheoVault"** (ή όποιο όνομα θέλετε)
3. Μετονομάστε το πρώτο φύλλο (tab) σε **Κατηγορίες**
4. Στη σειρά 1, γράψτε τις επικεφαλίδες:

   | A | B | C |
   |---|---|---|
   | id | name | parentId |

5. Δημιουργήστε ένα δεύτερο φύλλο (tab) με όνομα **Αντικείμενα**
6. Στη σειρά 1, γράψτε τις επικεφαλίδες:

   | A | B | C | D | E | F | G | H |
   |---|---|---|---|---|---|---|---|
   | id | name | category | location | description | photos | createdAt | updatedAt |

7. Δημιουργήστε ένα τρίτο φύλλο (tab) με όνομα **Translations** (προαιρετικό — δημιουργείται αυτόματα)
8. Στη σειρά 1, γράψτε τις επικεφαλίδες:

   | A | B | C |
   |---|---|---|
   | key | el | en |

> **Σημείωση:** Αν ξεχάσετε αυτό το βήμα, η εφαρμογή θα δημιουργήσει αυτόματα τα φύλλα με τις σωστές επικεφαλίδες. Το φύλλο Translations γεμίζει αυτόματα με τις προεπιλεγμένες μεταφράσεις όταν συνδεθεί editor.

## Βήμα 2: Εγκατάσταση Apps Script

1. Στο Google Sheet, πατήστε **Επεκτάσεις → Apps Script**
2. Θα ανοίξει ο επεξεργαστής κώδικα του Apps Script
3. Σβήστε ό,τι κώδικα υπάρχει (π.χ. `function myFunction(){}`)
4. Ανοίξτε το αρχείο `apps-script.js` με ένα text editor
5. Αντιγράψτε **όλο** τον κώδικα και επικολλήστε τον στον επεξεργαστή
6. Πατήστε **Ctrl+S** για αποθήκευση

## Βήμα 3: Δημοσίευση Web App

1. Στον επεξεργαστή Apps Script, πατήστε **Ανάπτυξη → Νέα ανάπτυξη**
2. Στο παράθυρο που ανοίγει:
   - **Τύπος:** Εφαρμογή web (Web app)
   - **Εκτέλεση ως:** Εσάς (τον λογαριασμό σας)
   - **Ποιος έχει πρόσβαση:** Οποιοσδήποτε (Anyone)
3. Πατήστε **Ανάπτυξη**
4. Θα σας ζητηθεί εξουσιοδότηση — πατήστε **Εξουσιοδότηση πρόσβασης** και ακολουθήστε τα βήματα
5. **Αντιγράψτε τη διεύθυνση URL** της εφαρμογής web (μοιάζει με: `https://script.google.com/macros/s/AKfycb.../exec`)

> **Σημαντικό:** Κρατήστε αυτό το URL! Το χρειάζεστε στο Βήμα 5.

## Βήμα 4: Δημιουργία Google OAuth Client ID

Αυτό χρειάζεται για τη σύνδεση Google (sign-in) στην εφαρμογή.

1. Ανοίξτε το [Google Cloud Console](https://console.cloud.google.com/)
2. Δημιουργήστε ένα νέο Project (ή χρησιμοποιήστε υπάρχον):
   - Πατήστε **Select a project → New Project**
   - Όνομα: **ArcheoVault** → **Create**
3. Ενεργοποιήστε το **OAuth consent screen**:
   - Αριστερό μενού → **APIs & Services → OAuth consent screen**
   - Επιλέξτε **External** → **Create**
   - Συμπληρώστε μόνο τα υποχρεωτικά:
     - App name: **ArcheoVault**
     - User support email: *το email σας*
     - Developer contact: *το email σας*
   - Πατήστε **Save and Continue** σε κάθε βήμα
   - Στο τέλος πατήστε **Back to Dashboard**
4. Δημιουργήστε **OAuth Client ID**:
   - Αριστερό μενού → **APIs & Services → Credentials**
   - Πατήστε **Create Credentials → OAuth client ID**
   - Application type: **Web application**
   - Name: **ArcheoVault Web**
   - Authorized JavaScript origins: προσθέστε τα domains που θα χρησιμοποιήσετε, π.χ.:
     - `https://archeovault.netlify.app` (αν φιλοξενείτε σε Netlify)
     - `https://yourusername.github.io` (αν φιλοξενείτε σε GitHub Pages)
     - `http://localhost` (για τοπική δοκιμή)
   - Πατήστε **Create**
5. **Αντιγράψτε το Client ID** (μοιάζει με: `123456789-abc...xyz.apps.googleusercontent.com`)

> **Σημείωση:** Αν ανοίγετε το αρχείο τοπικά (file://), η σύνδεση Google δεν λειτουργεί. Χρησιμοποιήστε Netlify, GitHub Pages ή localhost server.

## Βήμα 5: Σύνδεση της Εφαρμογής

1. Ανοίξτε το αρχείο `index.html` με text editor (π.χ. Notepad, VS Code)
2. Βρείτε τις γραμμές στην αρχή:
   ```javascript
   const API_URL = '';
   const GOOGLE_CLIENT_ID = '';
   ```
3. Συμπληρώστε τις τιμές:
   ```javascript
   const API_URL = 'https://script.google.com/macros/s/AKfycb.../exec';
   const GOOGLE_CLIENT_ID = '123456789-abc...xyz.apps.googleusercontent.com';
   ```
4. Αποθηκεύστε το αρχείο
5. Ανεβάστε στο Netlify/GitHub Pages ή ανοίξτε τοπικά

## Βήμα 6: Ρύθμιση Editors (ποιοι μπορούν να επεξεργάζονται)

Η εφαρμογή δημιουργεί αυτόματα ένα φύλλο **"Editors"** στο Google Sheet σας με τον ιδιοκτήτη ως πρώτο editor.

Για να προσθέσετε editors:
1. Ανοίξτε το Google Sheet
2. Πατήστε στο tab **Editors**
3. Προσθέστε τα Gmail addresses (ένα ανά σειρά στη στήλη A, κάτω από τον header)

| A |
|---|
| email |
| user1@gmail.com |
| user2@gmail.com |

**Χωρίς σύνδεση:** Οποιοσδήποτε μπορεί να βλέπει τα αντικείμενα (read-only).
**Με σύνδεση (editor):** Μπορεί να προσθέτει, τροποποιεί και διαγράφει αντικείμενα και κατηγορίες.

## Βήμα 7: Κοινή Χρήση

1. Στείλτε στον δεύτερο χρήστη τη **διεύθυνση URL** της σελίδας (π.χ. `archeovault.netlify.app`)
2. Αν θέλετε να έχει δικαίωμα επεξεργασίας:
   - Προσθέστε το email του στο φύλλο **Editors**
   - Πρέπει να πατήσει **Sign in with Google** στην εφαρμογή

## Φωτογραφίες (μέσω Google Drive)

1. Ανεβάστε τις φωτογραφίες σε ένα φάκελο στο Google Drive
2. Για κάθε φωτογραφία: δεξί κλικ → **Λήψη συνδέσμου** → **Ανοίξτε πρόσβαση σε όλους**
3. Αντιγράψτε τον σύνδεσμο και επικολλήστε τον στο πεδίο φωτογραφίας του αντικειμένου
4. Η εφαρμογή θα μετατρέψει αυτόματα τον σύνδεσμο σε εικόνα

## Αντίγραφο Ασφαλείας (Backup)

- Χρησιμοποιήστε το κουμπί **Εξαγωγή / Εισαγωγή** στην εφαρμογή για να κατεβάσετε ένα αρχείο JSON
- Για επαναφορά (μόνο editors), χρησιμοποιήστε το ίδιο κουμπί για εισαγωγή του αρχείου JSON

## Ενημέρωση του Apps Script

Αν χρειαστεί ενημέρωση του κώδικα:

1. Ανοίξτε το Apps Script (Επεκτάσεις → Apps Script στο Sheet)
2. Αντικαταστήστε τον κώδικα
3. Πατήστε **Ανάπτυξη → Διαχείριση αναπτύξεων**
4. Στην ενεργή ανάπτυξη πατήστε **Επεξεργασία** (μολύβι)
5. Αλλάξτε την **Έκδοση** σε "Νέα έκδοση"
6. Πατήστε **Ανάπτυξη**

Το URL παραμένει το ίδιο — δεν χρειάζεται αλλαγή στο `index.html`.

## Γλώσσα / Μεταφράσεις

Η εφαρμογή υποστηρίζει **Ελληνικά (EL)** και **Αγγλικά (EN)**. Ο χρήστης μπορεί να αλλάξει γλώσσα πατώντας το κουμπί **EN/EL** στην κεφαλίδα.

Οι μεταφράσεις αποθηκεύονται στο φύλλο **Translations** του Google Sheet:

| key | el | en |
|-----|----|----|
| btn.save | Αποθήκευση | Save |
| btn.cancel | Ακύρωση | Cancel |
| ... | ... | ... |

- Την πρώτη φορά που θα συνδεθεί editor, οι προεπιλεγμένες μεταφράσεις γράφονται αυτόματα στο φύλλο
- Για να αλλάξετε κάποιο κείμενο, ανοίξτε το Google Sheet → tab **Translations** → τροποποιήστε τη στήλη `el` ή `en`
- Η αλλαγή εμφανίζεται αμέσως μετά από ανανέωση της σελίδας
- Η γλώσσα αποθηκεύεται τοπικά (localStorage) — κάθε χρήστης κρατά τη δική του προτίμηση

## Αντιμετώπιση Προβλημάτων

| Πρόβλημα | Λύση |
|----------|------|
| "API not configured" | Βεβαιωθείτε ότι συμπληρώσατε `API_URL` και `GOOGLE_CLIENT_ID` στο `index.html` |
| Σφάλμα CORS | Βεβαιωθείτε ότι η ανάπτυξη είναι "Οποιοσδήποτε" (Anyone) |
| Google Sign-in δεν εμφανίζεται | Ελέγξτε ότι το domain είναι στα Authorized JavaScript origins |
| Sign-in αποτυγχάνει από file:// | Χρησιμοποιήστε Netlify/GitHub Pages ή localhost server |
| "Δεν έχετε δικαίωμα επεξεργασίας" | Προσθέστε το Gmail στο φύλλο **Editors** |
| Δεν εμφανίζονται φωτογραφίες | Ελέγξτε ότι ο σύνδεσμος Drive είναι δημόσιος |
| Αλλαγές δεν φαίνονται | Ανανεώστε τη σελίδα (F5) — δεν υπάρχει αυτόματη ανανέωση |
| Σφάλμα εξουσιοδότησης | Ξαναδημοσιεύστε το Apps Script (Βήμα 3) |
